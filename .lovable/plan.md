

# خطة: واجهتين مختلفتين (موقع ويب + تطبيق موبايل)

## الفكرة
التطبيق حالياً مصمم بالكامل بشكل موبايل (Bottom Navigation، شاشة كاملة، بدون Sidebar). المطلوب هو إنشاء نظام يكتشف تلقائياً إذا المستخدم على متصفح ويب (ديسكتوب) أو على تطبيق الموبايل (APK)، ويعرض واجهة مناسبة لكل حالة.

---

## الفرق بين الواجهتين

| العنصر | موقع الويب (Desktop) | تطبيق الموبايل (APK) |
|--------|---------------------|----------------------|
| التنقل | Sidebar جانبي ثابت | Bottom Navigation سفلي |
| الهيدر | شريط علوي عريض مع لوغو + روابط | الهيدر الحالي المدمج |
| المحتوى | أعمدة متعددة (Grid) مع مساحات أوسع | عمود واحد بتصميم موبايل |
| Footer | Footer كامل مع روابط | بدون Footer (مساحة للـ Bottom Nav) |

---

## خطوات التنفيذ

### 1. إنشاء Hook لكشف نوع الجهاز
ملف `src/hooks/useDeviceType.ts` يكتشف إذا المستخدم على:
- **ديسكتوب** (عرض الشاشة اكبر من 1024px)
- **موبايل** (عرض أقل أو داخل Capacitor)

يستخدم Capacitor API (`Capacitor.isNativePlatform()`) للتأكد إذا التطبيق يشتغل كـ APK.

### 2. إنشاء تخطيط الويب (Desktop Layout)
ملف `src/components/layout/WebLayout.tsx`:
- **Sidebar** جانبي ثابت فيه كل الروابط (الرئيسية، الإشارات، المجتمع، الأخبار، الخدمات، التحليلات، الملف الشخصي)
- **هيدر علوي** عريض مع اللوغو والإشعارات ومبدل السمة
- **منطقة محتوى** أوسع مع padding مناسب للشاشات الكبيرة
- **Footer** يحتوي حقوق النشر وروابط مفيدة

### 3. تعديل AppLayout الحالي
تحويل `AppLayout` لمكون ذكي يقرر تلقائياً:
- إذا **ديسكتوب/ويب** ← يستخدم `WebLayout`
- إذا **موبايل/APK** ← يستخدم التصميم الحالي (Bottom Nav)

### 4. تحسين الصفحات للشاشات الكبيرة
- إضافة `max-width` و `container` للمحتوى
- تحويل القوائم من عمود واحد إلى Grid (2-3 أعمدة) على الديسكتوب
- تحسين HomePage لعرض الإشارات والمقالات والتحليلات بشكل شبكي

### 5. تحسينات CSS
إضافة responsive utilities في `index.css`:
- أنماط خاصة بالـ Sidebar
- تحسين المسافات والأحجام للشاشات الكبيرة
- إخفاء عناصر الموبايل على الديسكتوب والعكس

## التفاصيل التقنية

### الملفات الجديدة
- `src/hooks/useDeviceType.ts` - كشف نوع الجهاز
- `src/components/layout/WebLayout.tsx` - تخطيط الويب بالكامل
- `src/components/layout/WebSidebar.tsx` - القائمة الجانبية

### الملفات المعدّلة
- `src/components/layout/AppLayout.tsx` - إضافة منطق التبديل
- `src/pages/HomePage.tsx` - تحسين Grid للديسكتوب
- `src/index.css` - أنماط responsive جديدة

### منطق الكشف (مبسط)
```text
هل التطبيق يشتغل داخل Capacitor (APK)؟
  ├── نعم → واجهة موبايل (Bottom Nav)
  └── لا → هل عرض الشاشة > 1024px؟
        ├── نعم → واجهة ويب (Sidebar)
        └── لا → واجهة موبايل (Bottom Nav)
```

### شكل واجهة الويب (تقريبي)
```text
+------------------+----------------------------------------+
|                  |  Header (Logo + Notifications + Theme)  |
|    Sidebar       +----------------------------------------+
|                  |                                        |
|  - الرئيسية     |                                        |
|  - الإشارات     |         محتوى الصفحة                   |
|  - المجتمع      |         (Grid layout)                   |
|  - الأخبار      |                                        |
|  - الخدمات      |                                        |
|  - التحليلات    |                                        |
|  - الملف الشخصي |                                        |
|                  |                                        |
|                  +----------------------------------------+
|                  |  Footer (حقوق النشر)                   |
+------------------+----------------------------------------+
```

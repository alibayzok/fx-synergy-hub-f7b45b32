

# خطة تحسين صفحة الخدمات بشكل احترافي

## نظرة عامة

سنقوم بإعادة هيكلة صفحة الخدمات لتصبح أكثر احترافية وجاذبية، مع التركيز على:
1. **خدمة الوسيط (Broker)**: صفحة هبوط احترافية لاستقطاب المتداولين للتسجيل تحت الوكالة
2. **خدمة USDT**: نظام متكامل لطلبات البيع والشراء مع واجهة احترافية

---

## المرحلة 1: إعادة هيكلة صفحة الخدمات الرئيسية

### التغييرات المقترحة على `ServicesPage.tsx`:

**قبل:**
- بطاقات بسيطة للخدمات
- نموذج طلب واحد للجميع

**بعد:**
- 3 تبويبات رئيسية: **الخدمات** | **طلباتي** | **USDT Exchange**
- كل خدمة لها صفحة/قسم مخصص

## المرحلة 4: تحسين نموذج طلب USDT

### حقول النموذج المحسّن:

| الحقل | النوع | مطلوب | الوصف |
|-------|------|-------|-------|
| نوع العملية | Select | ✅ | شراء / بيع |
| المبلغ | Number | ✅ | المبلغ بالـ USDT |
| الشبكة | Select | ✅ | TRC20 / ERC20 / BEP20 |
| طريقة الدفع | Select | ✅ | تحويل بنكي / OMT / كاش... |
| عنوان المحفظة | Text | للشراء | عنوان استلام USDT |
| ملاحظات | Textarea | ❌ | أي تفاصيل إضافية |

### تحديث قاعدة البيانات:

إضافة حقول جديدة لجدول `service_requests`:

```sql
ALTER TABLE public.service_requests
ADD COLUMN payment_method TEXT,
ADD COLUMN wallet_address TEXT;
```

## التفاصيل التقنية

### 1. Migration لتحديث جدول الطلبات:

```sql
-- إضافة حقول جديدة لطلبات USDT
ALTER TABLE public.service_requests
ADD COLUMN IF NOT EXISTS payment_method TEXT,
ADD COLUMN IF NOT EXISTS wallet_address TEXT;
```

### 2. تحديث الـ Hook:

```typescript
// useServiceRequests.tsx
const createRequest = async (data: {
  type: ServiceType;
  amount?: number;
  network?: string;
  payment_method?: string;  // جديد
  wallet_address?: string;   // جديد
  notes?: string;
}) => { ... }
```

### 3. بنية الصفحة الجديدة:

```typescript
// ServicesPage.tsx
type Tab = 'broker' | 'usdt' | 'requests';

return (
  <AppLayout>
    <Header />
    <Tabs>
      <Tab value="broker">الوسيط</Tab>
      <Tab value="usdt">USDT</Tab>
      <Tab value="requests">طلباتي</Tab>
    </Tabs>
    
    {activeTab === 'broker' && <BrokerLandingSection />}
    {activeTab === 'usdt' && <UsdtServiceSection />}
    {activeTab === 'requests' && <RequestsSection />}
  </AppLayout>
);
```

## النتيجة المتوقعة

1. **صفحة وسيط احترافية** تجذب المتداولين للتسجيل تحت الوكالة
2. **قسم USDT متكامل** مع نموذج طلب شامل وإعلانات الأدمن
3. **تجربة مستخدم محسّنة** مع تبويبات واضحة ونماذج سهلة
4. **تصميم جذاب** مع ألوان وتدرجات احترافية
